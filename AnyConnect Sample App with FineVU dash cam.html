How To Use AnyConnect Sample App with FineVU dash cam:

    FineVU Camera Setup for the developers:
    • Unzip AnyConnectCameraApp.zip on a computer and plug the FineVU SD card on the USB
    • Copy all the content of AnyConnectCameraApp folder to the root directory of the FineVU memory card
    • Safely eject the memory card and plug the card to the FineVU camera
    • Plug the USB-UART to the camera and the computer and run a screen session with the baud rate 115200 to access the camera
    • On the camera console go to the directory /tmp/SD1/bin

      • Execute the StreamApp

    FineVU Camera Setup for the users:
    • Insert the SD card to the camera which has the AnyConnect platform software.
    • Power on the FineVU Dashcam
    • Wait for 2 minutes to complete the boot-up

      • Now pair the camera using the AnyConnect Android Sample App

    Launch AnyConnect Android Sample App and Onboard with below details:
    • Endpoint Type: App
    • Mechanism: Field
    • Endpoint Name: YourName-CompanyName

      • Then click Onboard

    Grant all required Permissions for AnyConnect Sample App:
    Successfully Onboarded, and Login with Google:
    Onboard FineVU camera using WiFi-AP:
    • Click on WiFi-AP
    • It will automatically SCAN and list the available access points
    • Select the access point FineVU_pal and put the password 1234567890
    • Then press OK
    • Onboard FineVU_pal pops-up
    • Put the WiFi access point SSID and Password which has Internet
    • Put the IP 192.168.1.42 and port 8080 and press Yes
    • Then if the FineVU camera gets the internet the app will be disconnected from the WiFi-AP connection.

      • Now close the AnyConnect Android app and reconnect the Mobile with the WiFi access point which has Internet and will see AnyConnect Smart Camera on ACL tab

      Then go to the STREAM tab of the android App and select the camera to stream.
